# Karo Pitch Demo Homepage

This project is a demo landing page for the `Karo Pitch` intern assignment by `KaroStartup`.

## Project Summary

The homepage is designed to present Karo Pitch as a clean, investor-focused platform where early-stage founders across India, especially from Bharat, Tier-2, and Tier-3 cities, can pitch their startups and connect with investors.

The design focuses on:

- clean modern UI
- startup ecosystem positioning
- easy navigation
- mobile-friendly layout
- subtle animations
- investor and founder credibility cues

## Tech Stack

- React
- Vite
- CSS

## Local Setup

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```
